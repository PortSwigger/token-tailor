Token Tailor is a Burp Suite Community Edition extension that aims to simplify security testing by automating authentication (JWT and Basic) renewal.

This extension eliminates the need for manual token extraction and updates, reducing errors caused by expired tokens during scans and streamlining your workflow.

Configure Token Tailor to handle session renewals for all traffic passing through Burp Suite, providing a "tailor-made" solution for efficient session management.

Key Features:

  • Automate JWT and Basic authentication token renewals.
  • Seamless integration with other tools like Burp Scanner and SQLMap.
  • Easy to complex request flow configuration for retrieving authentication tokens.
  • Multiple and customizable expiration condition handling.
  • Import/export settings for faster configuration setup.
